# Expected Output: PDF Handout

The PDF handout demo generates a styled PDF file.

## Live Demo Result

When the demo runs successfully, it creates a PDF file at:
```
~/Desktop/automated/handout-5-essential-git-commands-YYYYMMDD.pdf
```

## Fallback

If the live demo fails, you can:

1. **Show the skill structure:**
   Open `example-skills/pdf-handout-creator/` and walk through:
   - `SKILL.md` - The skill instructions
   - `create-handout.py` - The Python script that generates PDFs

2. **Describe the expected output:**
   - US Letter size (8.5" x 11")
   - Warm Cream (#F5F3EB) background
   - IBM Plex Sans headings
   - JetBrains Mono for code
   - 2px black borders, sharp corners
   - Clear visual hierarchy

3. **Show a pre-generated example:**
   If you've run the demo before the workshop, save a copy of the PDF here.
