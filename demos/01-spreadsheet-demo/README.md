# Demo 1: Spreadsheet Generation

**Duration:** ~3 minutes

## What This Demo Shows

- Claude Code can generate structured data from natural language
- Creates properly formatted CSV files
- Generates realistic, varied sample data

## Live Demo Prompt

Use the prompt in `prompt.txt` — copy and paste it into Claude Code.

## Expected Behavior

1. Claude analyzes the request
2. Generates 20 rows of realistic sales data
3. Asks permission to create the file
4. Saves `sales-q1-2025.csv` to current directory

## Fallback

If the live demo fails, show `expected-output/sales-q1-2025.csv`

## Talking Points

- **No coding required** — just describe what you need
- **Realistic data** — Claude understands context (dates in Q1, varied products, regional distribution)
- **Proper formatting** — correct CSV structure, date formats, numeric precision
